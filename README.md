# LearnFast - AI-Powered Learning Platform

[![Nuxt UI](https://img.shields.io/badge/Made%20with-Nuxt%20UI-00DC82?logo=nuxt&labelColor=020420)](https://ui.nuxt.com)

LearnFast is a kid-friendly, vibrant, and AI-powered learning platform designed to make education an adventure. Students can explore themed missions, earn rewards, and even generate their own custom learning experiences using advanced AI.

## Key Features

- 🧪 **AI Adventure Laboratory**: Create personalized learning adventures using an alchemical wizard.
- 📚 **AI Lesson Engine**: Generate lessons from simple prompts or file uploads (PDF/Images).
- 🏆 **Gamification**: Earn Atoms and Magic Coins, collect badges, and climb the leaderboard.
- 🎨 **Premium UI**: Vibrant, high-contrast design with playful micro-animations and custom typography.
- 📱 **Mobile First**: Fully responsive dashboard experience for learning on any device.

## Tech Stack

- **Framework**: Nuxt 4 (Composition API)
- **UI Library**: Nuxt UI (@nuxt/ui)
- **Styling**: Tailwind CSS
- **Icons**: Phosphor Icons (i-ph-*)
- **Typography**: Sour Gummy (Google Fonts)

## Setup

Make sure to install the dependencies:

```bash
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
pnpm dev
```

## Production

Build the application for production:

```bash
pnpm build
```

Locally preview production build:

```bash
pnpm preview
```

Check out the [Nuxt documentation](https://nuxt.com) for more information.
